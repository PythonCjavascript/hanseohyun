# ⚾ KBO 팀 시즌 득점 예측 모델 (OLS 회귀분석)
### (KBO Team Season Run Prediction using OLS Regression)
## 📌 프로젝트 개요 (Overview)
야구에서 승리하려면 점수(득점)를 내야 합니다. 그렇다면 **어떤 공격 지표가 득점에 가장 큰 영향을 미칠까요?**

이 프로젝트는 KBO 구단들의 시즌별 공격 데이터(안타, 홈런, 볼넷 등)를 독립변수로 사용하여, 해당 시즌의 **총 득점(Run)**을 예측하는 다중 선형 회귀(Multiple Linear Regression) 모델을 구축한 프로젝트입니다.

단순한 예측을 넘어, 통계적 검정(t-test, p-value)을 통해 **승리에 실질적으로 기여하는 변수와 그렇지 않은 변수**를 구분해 내는 데 초점을 맞췄습니다.

## 🎯 분석 목표 (Objectives)
1.  **득점 결정 요인 파악:** 안타, 홈런, 볼넷, 도루, 삼진 중 득점과 가장 강력한 선형 관계를 가지는 변수는 무엇인가?
2.  **예측 모델 구축:** 주요 스탯만으로 팀의 시즌 총 득점을 얼마나 정확하게(설명력, $R^2$) 맞출 수 있는가?
3.  **통계적 유의성 검증:** "삼진(SO)을 많이 당하면 정말 득점이 낮아질까?"와 같은 통념을 통계적으로 검증.

## 🛠 분석 방법론 (Methodology)

### 1. 데이터 수집 및 변수 설정
* **Data Source:** KBO 공식 기록 및 야구 통계 사이트 (LG Twins 등 주요 구단 데이터 포함)
* **종속 변수 (Dependent Variable):** $R$ (시즌 총 득점)
* **독립 변수 (Independent Variables):**
    * $H$ (안타)
    * $HR$ (홈런)
    * $BB$ (볼넷)
    * $SB$ (도루)
    * $SO$ (삼진)

### 2. 모델링 (Modeling)
* **Algorithm:** OLS (Ordinary Least Squares, 최소제곱법)
* **Tools:** Python (`pandas`, `statsmodels` or `scikit-learn`)

## 📊 주요 분석 결과 (Key Findings)

### 1. 모델 성능
* **결정계수 ($R^2$):** **0.938**
    * 이 모델은 KBO 팀 득점 변동의 **93.8%**를 설명할 수 있는 매우 높은 예측력을 보였습니다.

### 2. 변수의 유의성 (Interpretation)
* **유의미한 변수:** 안타($H$), 홈런($HR$), 볼넷($BB$), 도루($SB$)
    * P-value < 0.05 로 득점에 통계적으로 유의미한 양(+)의 영향을 미침.
* **유의미하지 않은 변수:** **삼진($SO$)**
    * 분석 결과 삼진의 P-value는 **0.733**으로 나타남.
    * **인사이트:** 삼진을 당하는 것 자체는 득점 창출력에 통계적으로 유의미한 악영향을 주지 않음. (공격적인 스윙의 결과일 수 있음)
